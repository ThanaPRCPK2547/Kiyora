# Target Variable Definition: `brand_primary`

## Target Variable คืออะไร?
ตัวแปรเป้าหมาย (Target Variable) สำหรับโปรเจกต์การวิเคราะห์ข้อมูลนี้คือ **`brand_primary`** ซึ่งดึงมาจากคำถามหลักในแบบสอบถามที่ว่า *"ปัจจุบันคุณใช้คลีนซิ่งแบรนด์ใดบ่อยที่สุด (เลือกเพียงคำตอบเดียว)"*

## เหตุผลที่เลือก Target นี้
1. **Business Impact (ผลกระทบเชิงธุรกิจและกลยุทธ์):** การระบุแบรนด์หลักที่ผู้บริโภคนิยมใช้มากที่สุด (Primary Brand) จะทำให้ทราบถึงสภาพการแข่งขันในตลาด (Market Share) และพฤติกรรมการตัดสินใจเลือกซื้อจริงของผู้บริโภค
2. **Predictive Analytics (การสร้างแบบจำลองคาดการณ์):** การใช้ Machine Learning เพื่อทำนายตัวแปร `brand_primary` จากตัวแปรอิสระ (Features) เช่น ข้อมูลประชากรศาสตร์ (อายุ, รายได้), สภาพผิว (Skin Type), และปัจจัยที่ให้ความสำคัญ (Factors) จะช่วยตอบคำถามว่า *"ผู้บริโภคที่มีโปรไฟล์และปัญหาผิวลักษณะนี้ มักจะตัดสินใจเลือกใช้แบรนด์ใด"*
3. **Actionable Insights:** ผลการทำนายและ Feature Importance สามารถนำไปต่อยอดในการวางแผนการตลาด หรือแม้แต่พัฒนาคลีนซิ่งสูตรใหม่ (Product Development) เพื่อดึงดูดลูกค้าจากแบรนด์คู่แข่งได้

## กระบวนการ Data Labeling และจัดการ Missing Values
- **ข้อมูลที่มีอยู่แล้ว:** ข้อมูลชุดนี้มี Ground Truth หรือผลลัพธ์ของ `brand_primary` จากผู้ตอบแบบสอบถามโดยตรง จึงไม่ต้องนิยามกฎในการสร้าง Label ใหม่
- **การจัดการกับ Missing Values:** ในกรณีของผู้ที่ระบุว่า "ไม่ใช้คลีนซิ่งแบบน้ำ" ซึ่งส่งผลให้คำตอบในข้อ `brand_primary` กลายเป็นค่าว่าง (NaN) ใน Pipeline การจัดการข้อมูล เราได้เลือกใช้อัลกอริทึม **RandomForestClassifier** ในการพยากรณ์และเติมเต็ม (Imputation) แบรนด์ที่มีความเป็นไปได้สูงสุดที่กลุ่มเป้าหมายเหล่านี้จะเลือกใช้ หากพวกเขาตัดสินใจซื้อคลีนซิ่ง
